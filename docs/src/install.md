# Installation

## Prerequisite
In order to use the **Gnuplot.jl** package you'll need [gnuplot](http://gnuplot.info/) (ver. >= 5.0) installed on your system, and its executable available in your path.


## Installation

In the Julia REPL type:
``` julia
using Pkg
Pkg.add("Gnuplot")
using Gnuplot
```
